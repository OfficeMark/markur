## Architecture pointers (where things live)
- **Boot chain:** `App.tsx` → `AuthProvider` (id-gates user/profile) → `PermissionsProvider` (grants, once) → `ActionHintsProvider`.
- **Canvas:** `FloorPlanCanvas.tsx` sizes deterministically — ResizeObserver on the container + intrinsic dims from the decoded PDF/image, fit in pixels. DB stores **no** plan dims (`width_px`/`height_px` NULL); intrinsic always comes from the decoded asset — don't add a dimensions backfill expecting it to change rendering (no code path reads those columns for sizing).
- **Floor toolbar:** `Floor.tsx` — Row 1 (breadcrumb + Map/Grid + desktop-only filter cluster) / mobile-only name-search row / Row 2 action grid. Lock toggle: `PlanSettingsMenu.tsx` (`useSetFloorPinsLocked`). Single-pin lock: `AssetDrawer.tsx` `LockBar` (`useUpdateAsset`, optimistic).
